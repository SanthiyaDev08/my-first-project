2D Dino Game (Java)

This is a simple 2D Dino running game made using Java Swing.

Features

* Dino can jump using SPACE
* Random obstacles
* Score system
* Game over on collision
* Press R to restart

How to Run

1. Compile the program

javac Main.java

2. Run the game

java Main

Controls

SPACE → Jump
R → Restart

Language Used

Java
